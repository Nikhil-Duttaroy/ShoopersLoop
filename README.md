# Shopic : E-commerce React App

#### Technologies Used
- React Js
- Redux (Sagas)
- Firebase
- Styled Components
- Stripe(with nodejs and express)
-  heroku

#### Features
- User Authentication with Persistance
- Cart Stored for each individual User
- Responsive Design
- Progressive Web App
