# ShoppersLoop :Voice Controlled E-commerce Web App

#### Technologies Used
- React Js
- Redux (Sagas)
- Firebase
- Styled Components
- Stripe(with nodejs and express)
- Heroku
- Alan AI

#### Features
- User Authentication with Persistance
- Cart Stored for each individual User
- Responsive Design
- Progressive Web App
- Voice Controller
